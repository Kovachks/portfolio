Skills

Projects

GhostTruck is a MERN stack restaraunt application which includes a fully functional content management system. To pair with the content management system we allow users to place orders and check out using Stripe. Orders will then be pushed through to the kitchen in real time, without the need for refreshing, using websockets. Once a user completes their order a text message will be sent out upon order clearing by an admin.

GeoVane is a web application which combines the power of Google Directions and Google Maps API's with the OpenWeather API in order to plot weather related data across a users requested directional route. This app was originally created using standard HTML/CSS/Jquery but I have recently converted it to be a Node.js application.

K+ Tutoring is a website I built for a local tutoring business. This application was built in Node.JS and express with HTML/CSS/Jquery. The application allows a user to contact the business owner via email and displays all information that the business owner saw as relavent to potential customers.
