Projects

CHVenue is a fully functional mock site for a fictional music venue. The site was designed by CarbonHouse and replicated by me via photoshop images provided. It is built entirely with HTML/CSS/JS with a Bootstrap CSS framework.

GeoVane is a web application which combines the power of Google Directions and Google Maps API's with the OpenWeather API in order to plot weather related data across a users requested directional route. This app was originally created using standard HTML/CSS/Jquery but was recently converted to Node.js.

GhostTruck is a MERN stack restaraunt application which includes a fully functional content management system. To pair with the content management system we allow users to place orders and check out using the Stripe API. Orders will then be pushed through to the kitchen in real time, without the need for refreshing, using websockets. Once a user completes their order a text message will be sent out upon order clearing by an admin.
